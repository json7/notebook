####插件配置####
```
a.不同语言的jetbrains家族ide统一采用metarial plugin
b.手动下载该插件到本地磁盘目录
c.打开ide preferences ,找到plugin 项, 然后选择install from disk, 选择 b 下载好的插件jar文件
d.插件主题选择 darker theme (如果preferences设置成darker theme后,左侧文件列颜色不发生变化,那么在ide top menu的tools里在设置一下)
f.我采用的ide背景色均为黑色,因此在preferences->application&behavior->file colors 里删除无用的颜色
```

####字体####
```
a.字体采用ubuntu mono derivative power
b.size 15
c.line spacing 1.6
```
